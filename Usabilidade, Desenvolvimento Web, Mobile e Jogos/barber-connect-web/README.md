# BarberConnectFrontEnd
Segue abaixo os passos para rodar o front-end localmente
1. Baixe o repositório
2. Extraia em uma pasta
3. Clique com o botão direito na pasta e em abrir no terminal
4. Cole e rode o seguinte código
cd barberconnect
python -m http.server 5500
5. Entre no navegador em http://localhost:5500
